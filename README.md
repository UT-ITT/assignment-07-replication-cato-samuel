## PianoText Redesigning the Piano Keyboard for Text Entry

We implemented a version of PianoText in `implementation/PianoText.py`. Documentation can be found in documentation.md and the presentation in `presentation/PianoText_ITT.odp`

### How to use it

- Connect a Midi Keyboard to your PC via USB
- Alternatively you can use an App like MIDIPadidi on your Android Smartphone, connect that via USB and set your Phone to transfer MIDI Signals
- start with `python implementation/PianoText.py`
- You can now use your Piano Keyboard to type
- For the keyboard mappings, consult the Live Demo Slide of the Presentation or Appendix A of the paper
- The program can be exited by pressing 3 Cs at the same time
- Pressing 4 keys at the same time will trigger backspace
- Pressing 5 keys at the same time will trigger space
